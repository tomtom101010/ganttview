Bedienungshilfe

  • Ansicht wechseln: Buttons "📊 Gantt-Ansicht" und "📋 Tabellen-Ansicht".
  • .gantt-Datei laden: Button "📁 .gantt Datei laden".
  • Projektauftrag in .gantt umwandeln: Anleitung.
  • Task bearbeiten (Gantt): Doppelklick auf Task oder Task auswählen + Enter.
  • Task bearbeiten (Tabelle): Doppelklick auf eine Zelle (außer "Ebene"). Enter/Klick außerhalb speichert, Esc bricht ab. Kalender für Daten, Dropdown für Farben.
  • Task verschieben/Größe ändern (Gantt): Task/Rand ziehen.
  • Reihenfolge ändern (Gantt): Task-Namen links ziehen (aktualisiert Tabelle).
  • Fortschritt ändern (Gantt): Im Menü oder Balken ziehen.
  • Verbinden (Gantt): Zwischen Punkten (⦿) ziehen. Doppelklick auf Linie löscht.
  • Speichern & Export:
    • 💾 .gantt speichern: Aktueller Projektstand.
    • 📄/🖼️ PDF/PNG Export: Aktuelle Gantt-Ansicht (nur dort aktiv).
    • 📄 Tabelle als CSV: Aktuelle Tabellendaten (nur dort aktiv).
  • Einbetten (iFrame):
    • "➕ Einbetten" klicken (nur in Gantt-Ansicht aktiv).
    • Code kopieren & in Webseite einfügen (zeigt nur Diagramm).
    • Benötigt öffentliches Hosting & Server-Erlaubnis.
  • Link teilen:
    • "🔗 Link teilen" klicken (nur in Gantt-Ansicht aktiv).
    • Link kopieren & versenden (zeigt Diagramm + Seite mit aktuellem Stand).
    • Benötigt öffentliches Hosting dieser Seite.
  • Tooltip (Gantt): Maus über Task-Balken.
  • Tabelle: Zeigt Tasks in Reihenfolge. Texte umgebrochen. Zellen per Doppelklick bearbeitbar.
